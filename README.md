# 《ROS机器人开发实践》源码

------
| 文件夹名        | 描述  |
| --------   | -----:  |
| ros_primary     | ROS基础功能 |
| robot_mrobot        |   mrobot相关的功能包   |
| robot_marm        |    marm相关的功能包    | 
| robot_perception        |    机器人感知相关的功能包    | 
| ros_advanced        |    ROS进阶功能    | 
| robot_learning        |    机器学习相关的功能包    | 
| ros2        |    ROS2功能包（不能放置在ROS1的工作空间下，否则会报错）    |
